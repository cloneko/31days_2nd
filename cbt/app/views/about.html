このシステムの開発について

このシステムは下記の技術/ツール/サービスを使って作っています

  • AngularJS — Superheroic JavaScript MVW Framework(JavaScriptのフレームワーク)
  • The web's scaffolding tool for modern webapps | Yeoman(Webアプリの雛形を作る奴)
  • CoffeeScript(JavaScriptを書き易くした奴)
  • GitHub(コードホスティング/バージョン管理)
  • Microsoft Visual Studio Code(エディタ)
  • クラウドコンピューティング なら アマゾン ウェブ サービス（AWS）(Webサーバーとかいろいろ)
